## 📊 Brand Guidelines

### Logo Usage
**DO**:
- Use on dark backgrounds (primary)
- Maintain minimum clear space
- Use provided color variants
- Scale proportionally

**DON'T**:
- Distort or stretch
- Change colors arbitrarily
- Add effects or shadows
- Use on busy backgrounds

### Color Usage
**Primary**: Use for CTAs, links, and accents
**Secondary**: Use for backgrounds and containers
**Neutral**: Use for text and borders

### Typography
**Headings**: Bold, large, gradient text for emphasis
**Body**: Regular weight, good line height
**Code**: Monospace font for technical content
